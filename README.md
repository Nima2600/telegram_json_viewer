# telegram_json_viewer
اسکریپتی جهت خواندن پیام و زمان ارسال پیام از طریق فایل JSON دریافت شده از طریق گیتهاب

شما میتوانید خروجی های تلگرام رو طبق میل خودتون تنظیم کنید.
این اسکریپت از telefeed برای دریافت فایل های json استفاده میکند و سپس با توجه به url پیام‌هارو با ساعت تهران از آخر به اول به شما نشان میدهد.
برای رد کردن یه کانال خاص میتوانید در زمان درخواست ورودی n را وارد کنید.

این اسکریپت صرفا جهت یادآوری پایتون و در فضای سرکوب اینترنتی به وجود آمده.
